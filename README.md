# TestIsh

add #ish helper to Numeric and Time

## Installation

Add this line to your application's Gemfile:

    gem 'test_ish'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install test_ish

## Usage

include TestIsh in your test setup or rspec config
